## 食用方式
1. 运行 Sheas Cealer -> 将文件 (镜像) 直链填入上游链接 -> 点击**更新上游规则**按钮获取更新 (首选)
2. 运行 Sheas Cealer -> 点击**编辑上游规则**按钮 -> 将相应内容复制到自动打开的文件中并保存 (临时使用)
